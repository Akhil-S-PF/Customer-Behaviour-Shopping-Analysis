# Customer-Behaviour-Shopping-Analysis
End-to-end data analytics project analyzing customer behaviour data using Python, PostgreSQL, and Power BI to generate actionable business insights.

## Overview

This project demonstrates an end-to-end **data analytics workflow**, starting from raw data processing to business insight generation. The objective is to analyze retail sales data to identify patterns, trends, and key business insights that can support data-driven decision making.

The project includes:

* Data loading and preprocessing using Python
* Exploratory Data Analysis (EDA)
* Running analytical SQL queries using PostgreSQL
* Building an interactive dashboard in Power BI
* Creating a project report and presentation

This project simulates a real-world **data analyst workflow** commonly used in business environments.

---

## Dataset

The dataset used in this project contains retail transaction data including information such as:

* Order details
* Product categories

The data is used to analyze:

* Sales performance
* Customer purchasing patterns

## Project Workflow

### 1. Data Loading

The dataset is loaded into Python using **Pandas** for further processing and analysis.

### 2. Data Cleaning

Data preprocessing steps include:

* Handling missing values
* Removing duplicates
* Correcting data types
* Creating new analytical columns

### 3. Exploratory Data Analysis (EDA)

EDA was performed to understand patterns and relationships in the data using visualizations such as:

* Sales distribution
* Revenue by category

Libraries used:

* Pandas
* Matplotlib
  

### 4. SQL Analysis

The cleaned dataset was imported into **PostgreSQL** to perform business analysis using SQL queries.

Key SQL analyses includes

* Top-selling products
* Customer segment revenue contribution

### 5. Dashboard Development

An interactive **Power BI dashboard** was built to visualize key metrics and insights.

Dashboard features include:

* Sales KPIs
* Category-wise performance
* Revenue trends
* Regional sales analysis
* Customer segmentation insights
